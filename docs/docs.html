Seamless Integration

Developers can integrate CareerForge in minutes using our typed SDKs. Our API is designed to be self-documenting and strictly typed.

POST /v1/matching/analyze
import { CareerForge } from '@careerforge/sdk'; // Initialize client const client = new CareerForge(process.env.API_KEY); // Run AI matching analysis const match = await client.matching.analyze({ candidateId: 'cand_8x92m...', jobId: 'job_v82n1...', weights: { experience: 0.6, skills: 0.4 } }); console.log(match.score); // 0.94
from careerforge import Client # Initialize client client = Client(api_key="cf_sk_...") # Run AI matching analysis match = client.matching.analyze( candidate_id="cand_8x92m...", job_id="job_v82n1...", weights={"experience": 0.6, "skills": 0.4} ) print(match.score) # 0.94
curl -X POST https://api.careerforge.com/v1/matching/analyze \ -H "Authorization: Bearer cf_sk_..." \ -H "Content-Type: application/json" \ -d '{ "candidate_id": "cand_8x92m...", "job_id": "job_v82n1..." }'

Developer Tip

Use the simulate flag in the sandbox environment to test matching scores without consuming credits.
